# Team-and-Employee-Leaves-Java
Simple StartUp Operation for City University of Hong Kong [CS2312 Java Assignment]

## How to Run the code
Please ensure to clone `Main.java` and all files inside the folder `"all-classes"`, as well as all `*.txt` files inside the `"testcases&outputs"` folder. When you run `Main.java`, it will prompt the user to input the path of a filename. You can enter the name of a file that exists among the `.txt` files. Then it will give you an output.
